Front-End Developer Digest

This is a page where developers can express their thoughts or ideas about various topics ranging from current web standards and best practices to anecdotes about landing a first gig. Anyone even mildly interested in tech is encouraged to follow and comment! Many developer blogs feature useful posts that include tips, tricks, and techniques for incorporating cool effects into websites using a bit of clever CSS and JavaScript in tandem. One of my first goals will be to point users towards these resources and create an archive.

Adding Articles -- June 10, 2016

Eventually, I'll have an organized method of adding articles to this page without directly altering the structure of the html. Until then, I'll be updating the html directly. This begs the question: How do bloggers (that code their own websites) typically organize the articles that they add? These articles usually have their own IDs corresponding to a particular entry or page. Consider this the next topic of my research!

Why all the hype?

This page is helping me develop my skills on the front and back end of web development. Before it goes live, we'll make sure that the layout, form submission, and interface is exactly as we've envisioned it. The three major hurdles to having an impressive live page are:

  • An appealing and user-friendly layout (with pictures!)
  • Easy-to-use interface for submitting comments and content (some backend required)
  • Several pages with content to fill them, content that a reader base will find useful.

How you can help
